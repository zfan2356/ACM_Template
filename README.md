# ACM算法竞赛常见模板(模板元编程实现)

本项目实现了ACM中部分经典算法的模板化, 为了更好地完成普适性, 避免对模板的修改,
将会使用模板元编程实现模板算法, 其余部分算法也会使用lambda语法实现
(部分模板参考jiangly)
